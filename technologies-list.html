Технологічний стек проекту "Forchetta"

Архітектура веб-додатку для замовлення кондитерських виробів

🖥️ 1. Frontend (Клієнтська частина)

Створення користувацького інтерфейсу, робота з браузером, маршрутизація по сторінках та управління станом.

  • React 18: основний фреймворк для побудови UI компонентами.
  • Vite: надшвидкий збирач проекту.
  • Tailwind CSS: стилізація сторінок прямо в класах (адаптивність, дизайн-система).
  • Radix UI: безстилеві доступні (accessible) компоненти (модальні вікна, дропдауни).
  • TanStack Query: кешування відповідей від сервера (швидке завантаження каталогу та кошика).
  • Zustand: глобальне сховище даних (авторизація юзера, активні фільтри пошуку).
  • Embla Carousel: легка та гнучка бібліотека для створення каруселей зображень (сторінка товару).
  • React Router v6: система навігації по сайту без перезавантаження сторінок (SPA).
  • Lucide React / SVG: векторна графіка та оптимізовані іконки інтерфейсу.
  • React Hot Toast: красиві спливаючі сповіщення (успішне додавання, помилки).

⚙️ 2. Backend (Серверна частина)

Мозковий центр проекту. Приймає запити, перевіряє безпеку (токени), рахує суми замовлень.

  • Node.js: серверне середовище виконання JavaScript.
  • Express.js: каркас для створення REST API маршрутів конролерів.
  • JWT / Passport.js: токени для безпечної авторизації та інтеграції Google OAuth.

🗄️ 3. Бази даних та Кешування

Надійне сховище замовлень, профілів та швидкий доступ до тимчасових даних (кошика).

  • MongoDB Atlas: хмарна NoSQL БД (зберігає товари, категорії, історію замовлень).
  • Redis: надшвидка In-Memory БД (тримає в пам'яті кошик юзера та тимчасові коди).
  • Mongoose: ORM інструмент для суворої типізації схем даних у MongoDB.

☁️ 4. Зовнішні сервіси (API Integrations)

Делегування складних задач спеціалізованим платформам.

  • Google Cloud Platform: авторизація в один клік без пароля.
  • Cloudinary: хмарне сховище (оптимізує та роздає фотографії десертів).
  • Stripe: платіжний шлюз для безпечної емуляції оплат з карток.
  • Resend: сучасний API для транзакційних email-розсилок (чеки, скидання пароля).

🌐 5. Інфраструктура та Деплой

Публікація сайту в інтернеті та захист мережі.

  • Railway PaaS: платформа для автоматичного розгортання додатку з GitHub.
  • Cloudflare: мережа доставки контенту (CDN), управління DNS та захист від DDoS атак.
